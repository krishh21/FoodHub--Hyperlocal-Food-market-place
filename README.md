## Description
FoodHub is built to solve the problem of finding local food options quickly and interactively. It includes features like product listings, cart management, search functionality, and responsive design.

---

## Features
- Functional and class components
- React Router v6 with dynamic routing and `<Outlet>`
- Global state management via Redux Toolkit
- Custom hooks for API interactions
- Tailwind CSS for utility-first styling
- Code-splitting using `React.lazy` and `Suspense`
- Controlled form inputs and best practices across the app

---

## Tech Stack
- **React** (v19)
- **Parcel** for bundling
- **Redux Toolkit** and **React-Redux**
- **Tailwind CSS**
- **Jest** for testing
- **Babel** for transpilation
